# 1024点FFT CUDA程序说明

## 概述
这是一个使用CUDA实现的1024点快速傅里叶变换(FFT)程序，包含自定义kernel实现和cuFFT库两种方法。

## 文件说明
- `fft.cu` - 主程序源代码

## 功能特性
1. **自定义FFT实现** - 基于Cooley-Tukey蝶形算法
   - 位反转排序
   - 多级FFT蝶形操作
   
2. **cuFFT库版本** - 使用NVIDIA官方优化库进行对比

3. **测试与验证**
   - 自动生成测试信号（混合正弦波）
   - 性能计时对比
   - 结果验证

## 编译要求
- CUDA Toolkit (支持compute capability 3.0+)
- NVCC编译器
- cuFFT库

## 编译命令
```bash
nvcc -o fft fft.cu -lcufft
```

## 运行
```bash
./fft
```

## 输出示例
程序将输出：
- 自定义FFT执行时间
- cuFFT库执行时间
- FFT结果的幅值和相位（前10个点）
- 两种方法的结果对比验证

## 性能说明
- FFT点数：1024
- 线程块大小：256
- 适用于信号处理、频谱分析等应用
